Первая команда чтобы изначально инициализировать гит в ваш проект
>>> git init 

Увидим >>> Initialized empty Git repository in /Users/mansur/Desktop/test_git_11_1_dp/.git/
Снова если ввести >>> Reinitialized existing Git repository in /Users/mansur/Desktop/test_git_11_1_dp/.git/


Нам нужно добавить файл в индекс git чтобы git отслеживал наш файл
>>> git add main.py
>>> git add .


После того как мы добавили файлы в отслеживание
нам нужно сделать снимок нашего проекта
>>> git commit -m 'Мы пишем любое названием, прост не пишите его как я sdf'

Теперь после этого сообщения мы отправляем наш проект которые отслеживает гит
В САМОМ ПЕРВОМ ОТПРАВКЕ ФАЙЛОВ С ГИТА
git branch -M main 
git remote add origin https://github.com/MovsarCoder/test_git_11_1_dp.git
 git push -u origin main


В дальнейшем когда вы снова хотите отправить файлы с гит в гитхаб 
>>> git push



Команда чтобы к себе на компьютер закачать мой проект
>>> git clone ссылка_которую_вы_копируете_на_репо


Ветки нужны для того чтобы не менять главныый код 
с главной Ветки
а создать второстепенную ветку с тем же самым кодом


Команда чтобы посмотреть на Ветки
>>> git branch

Создать новую ветку
git branch dp_11_1

Я нахожусь на данный момент на ветке main
я хочу перейти в ветку dp_11_1
Перемещение по веткам
>>> git checkout dp_11_1


А как мне в ЭТУ ветку получить данные с другой Ветки
>>> git merge название_ветки_с_которой_хотите_взять_изменения


Чтобы вытащить изменения с гитхаб проекта
>>> git pull
